Portfolio

The Car Doctor

Car Doctor is a full-stack web application designed to streamline the process of booking car repair and maintenance services. Users can register, log in, and schedule appointments with expert technicians. The platform features secure authentication using Firebase, dynamic routing with React Router, and real-time data management through MongoDB and Express.js. This project showcases proficiency in modern web development technologies and emphasizes user-friendly design and functionality.

View Project
NSU Shuttle Service UI

The NSU Shuttle Bus Service project is a frontend user interface designed to improve the campus transportation experience for North South University students. Built with clean HTML and CSS, the UI offers an intuitive layout for viewing shuttle schedules, routes, and availability. This project focuses on simplicity, responsive design, and user accessibility, serving as the foundation for a potential full-featured web app in the future.
